# Selenium Automation using Python

## Overview

This project demonstrates browser automation and data extraction using Selenium WebDriver in Python.

The scripts perform:

* Opening websites using Selenium
* Maximizing browser windows
* Searching data using input fields
* Clicking buttons and links
* Refreshing web pages
* Extracting multiple elements from a webpage
* Displaying extracted data in the terminal
* Closing the browser automatically

## Technologies Used

* Python 3
* Selenium WebDriver
* Google Chrome

## Features

### 1. Google Search Automation

* Opens Google
* Searches for "Python Selenium Tutorial"
* Displays search results page
* Refreshes page

### 2. Amazon Search Automation

* Opens Amazon India
* Searches for "Laptop"
* Clicks search button
* Refreshes page

### 3. Data Extraction

* Opens Amazon India
* Searches for "Headphones"
* Extracts product names using Selenium
* Displays extracted data in terminal
